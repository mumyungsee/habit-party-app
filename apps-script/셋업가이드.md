# 습관파티 시트 연동 셋업 가이드

> 이대로 따라 하면 됨. 막히면 단계 번호 말해줘.

## 1단계 — 시트 탭 (이미 Claude가 만들어 둠)

`members`, `checkins` 두 탭은 이미 생성·세팅 완료. 구조 참고용:

### `members` 탭
헤더: `id | name | team | role | mission | emoji | pin`
- 28개 역할 줄이 미리 채워져 있음 (팀/역할/임무/이모지)
- 사람들은 **자기가 맡을 역할 줄의 `name` 칸에 이름만** 적으면 됨 ("빈 자리 골라 앉기")
- `id`/`pin`은 비워둠 (앱·스크립트가 자동)

### `checkins` 탭
헤더: `memberId | day | date | done | memo | updatedAt`
- 비어있음. 앱이 인증할 때마다 자동으로 쌓음

---

## 2단계 — Apps Script 코드 붙이기

1. 시트 상단 메뉴 → **확장 프로그램 > Apps Script**
2. 열린 편집기의 기존 내용 다 지우고 → `Code.gs` 파일 내용 전체 복붙
3. 저장 (Ctrl+S)

---

## 3단계 — 웹앱으로 배포

1. 우상단 **배포 > 새 배포**
2. 톱니바퀴(유형 선택) → **웹 앱**
3. 설정:
   - 설명: 아무거나 (예: 습관파티)
   - **실행 계정: 나**
   - **액세스 권한: 모든 사용자**
4. **배포** 클릭 → (처음이면 권한 승인 팝업 뜸 → 본인 구글계정 허용)
5. 끝나면 **웹 앱 URL** 이 나옴 (`https://script.google.com/macros/s/.../exec`)
   → **이 URL을 복사해서 나한테 알려줘**

---

## 4단계 — 챌린지 시작일 설정

`Code.gs` 위쪽의 이 줄을 23기 실제 시작일로 바꾸기:
```
const CHALLENGE_START = "2026-06-02";  // ← 여기
```
바꾼 뒤 다시 저장 + (배포 > 배포 관리 > 수정 > 새 버전 배포)

---

## 이후 (내가 할 일)
- 너가 준 웹앱 URL을 앱 `data.js`에 꽂으면 → 진짜 시트랑 연결됨
- 그 뒤론: 사람들이 시트에 이름 적기 → 앱에서 본인 선택 → 핀 정하기(시트에 저장) → 인증(시트에 쌓임)
